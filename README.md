# 😊 Sentiment Analyzer Web App

A web app that analyzes the sentiment of any text and tells you if it is Positive, Negative, or Neutral!

## 🌐 Live Demo
[Click here to try the app!](https://sentiment-web-app-gtqmvklk5pd3srqskgjxrs.streamlit.app/)

## 📸 What it does
- User types any text
- App analyzes the sentiment
- Shows if it is Positive 😊, Negative 😠, or Neutral 😐
- Shows a confidence score

## 🛠️ Built With
- Python
- Streamlit
- TextBlob
- Natural Language Processing (NLP)

## 🚀 How to Run Locally
```bash
pip install streamlit textblob
streamlit run app.py
```
